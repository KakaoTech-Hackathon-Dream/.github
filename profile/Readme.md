# 드림, 당신의 그리운 꿈을 드립니다.

## 프로젝트 설명
생성형 AI(LLM, RAG, Diffusion model)를 활용하여 노년층이 이루지 못한 꿈을 바탕으로 ‘꿈을 이룬 나’에 대한 이야기와 이미지를 생성하고, 이를 통해 관련된 교육 프로그램 및 직업을 추천하는 서비스
